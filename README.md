# About Me
I'm a Linux-passionate technologist transitioning from cybersecurity into DevOps and platform engineering. With hands-on experience as a SOC Analyst, I developed a strong instinct for system behavior, threat detection, and network fundamentals — skills that make me a security-aware DevOps engineer from day one.

I'm currently deepening my expertise in the modern DevOps stack: building pipelines with GitLab CI/CD, containerizing workloads with Docker and Kubernetes, provisioning infrastructure with Terraform and Ansible, and managing environments with Vagrant and Packer.

My long-term goal is to grow into DevSecOps or Cloud Security Engineering — bridging the gap between fast-moving infrastructure and robust security practices.

📌 Open to DevOps, Platform Engineering, and SRE roles.
